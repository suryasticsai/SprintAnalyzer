# SprintAnalyzer · AgileAdvisor v0.5

**RAG‑powered Agile automation assistant** – built by Sai Varakala (@suryasticsai)

SprintAnalyzer is a modern web interface that integrates **AgileAdvisor v0.5**, a Retrieval-Augmented Generation (RAG) AI assistant designed to help Scrum Masters, Product Owners, and Agile teams answer questions about sprint metrics, retrospectives, story points, anti-patterns, and best practices. The platform also showcases a suite of complementary Agile intelligence tools.

**Live Demo** – [Deployed on Netlify / GitHub Pages]
🔗 https://sprintanalyzer.netlify.app
🔗 https://suryasticsai.github.io/SprintAnalyzer
---

## ✨ Key Features

- **AgileAdvisor v0.5** – RAG‑powered AI assistant (iframe integration with Hugging Face Space)
- **Dark / Light mode** – Persistent theme preference
- **Responsive design** – Works on desktop, tablet, and mobile
- **Tool ecosystem** – Quick links to ScrumLens, Alkembic, and documentation
- **Collaboration section** – Contact links to engage with the creator
- **Smooth scrolling & scroll‑to‑top** – Enhanced UX

- Vanilla JavaScript (theme toggling, smooth scroll, local storage)

## 🚀 Deployment Instructions

### Deploy on **GitHub Pages**

1. Fork / clone this repository.
2. Place the `index.html` file in the root directory (and any assets if needed).
3. Go to repository **Settings > Pages**.
4. Under **Branch**, select `main` (or `master`) and `/root` folder.
5. Click **Save**. Your site will be published at `https://<username>.github.io/<repo-name>`

### Deploy on **Netlify** (recommended for custom domains & forms)

1. Drag & drop the folder containing `index.html` onto [Netlify Drop](https://app.netlify.com/drop) – **or**
2. Connect your GitHub repository to Netlify.
3. Set **Publish directory** as `/` (root).
4. Netlify automatically builds and provides a live URL.
5. (Optional) Add a custom domain.

> **Note:** The embedded iframe points to `https://suryasticsai-agileadvisor.hf.space`. Ensure that URL is accessible in your deployment environment (it is publicly available).
